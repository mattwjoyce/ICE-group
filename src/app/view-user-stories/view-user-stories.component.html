User Story 1 "As a fan, I want the homepage to show a summary of relevant data to current week (ladder, round results, upcoming match)"
User Story 2 “As a fan, I want to see the prediction that my team will win their next game”
User Story 3 “As a fan, I want to see the results of all games my team has played this season”
User Story 4 “As a fan, I want to see opponent and games details for the next 5 matches my team will play”
User Story 5 “As a fan, I want to see all the venues where my team has won so far this season”
User Story 6 “As a fan, I want to see the head-to-head games and if available, results between my team and my team’s rival this season”
User Story 7 "As a fan, I want to see the highest amount of points my favourite team has scored in a game this season"
User Story 8 "As a fan, I want to see the results of the game this season which had the largest point difference"
